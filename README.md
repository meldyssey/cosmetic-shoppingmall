# 코스메틱 쇼핑몰 홈페이지 리뉴얼 프로젝트

## 📋 프로젝트 개요

프리미엄 코스메틱 브랜드의 공식 웹사이트 리뉴얼 프로젝트입니다. 사용자 친화적인 UI/UX와 효율적인 관리자 시스템을 통해 온라인 쇼핑 경험을 개선하였습니다.

### 주요 기능

#### 🛍️ 사용자 기능
- **제품 카탈로그**: 카테고리별 제품 브라우징 및 상세 정보 조회
- **장바구니 & 결제**: Toss Payments 연동을 통한 안전한 결제 시스템
- **회원 관리**: 회원가입, 로그인, 카카오 소셜 로그인
- **마이페이지**: 주문 내역, 개인정보 관리, 배송 조회
- **1:1 문의**: 고객 지원 시스템
- **리뷰 작성**: 제품 리뷰 및 평점 시스템
- **매장 찾기**: Kakao Map API를 활용한 매장 위치 검색

#### 👨‍💼 관리자 기능
- **대시보드**: 실시간 판매 통계 및 데이터 분석 (Chart.js)
- **회원 관리**: 회원 정보 조회 및 관리
- **주문 관리**: 주문 내역 조회 및 배송 상태 관리
- **제품 관리**: 제품 등록, 수정, 삭제 및 이미지 업로드
- **문의 관리**: 1:1 문의 답변 및 관리
- **리뷰 관리**: 제품 리뷰 모니터링 및 관리

### 기술 스택

#### Frontend
- **Framework**: React 18.3.1
- **상태 관리**: Redux, Redux-Thunk
- **라우팅**: React Router DOM 6.28.0
- **스타일링**: SCSS, Emotion
- **UI 라이브러리**:
  - Swiper (캐러셀)
  - Font Awesome (아이콘)
  - Chart.js & React-Chartjs-2 (데이터 시각화)
- **API 연동**:
  - Toss Payments SDK (결제)
  - React Kakao Maps SDK (지도)
  - React Daum Postcode (주소 검색)

#### Backend
- **Runtime**: Node.js
- **Framework**: Express 4.21.2
- **데이터베이스**: MySQL 2.18.1 / MySQL2 3.11.5
- **인증**: bcrypt 5.1.1
- **파일 업로드**: Multer 1.4.5
- **이메일**: Nodemailer 6.9.16
- **API 통신**: Axios 1.7.9

#### DevOps & Tools
- **버전 관리**: Git
- **환경 변수**: dotenv
- **CORS**: cors 2.8.5

## 🔒 보안 기능

- **비밀번호 암호화**: bcrypt를 사용한 안전한 비밀번호 해싱
- **CORS 설정**: 허용된 도메인만 API 접근 가능
- **환경 변수**: 민감한 정보는 .env 파일로 분리 관리
- **파일 업로드 제한**: 5MB 크기 제한 및 파일 타입 검증
- **SQL Injection 방지**: Prepared Statement 사용

## 📊 데이터베이스 스키마 (주요 테이블)

- **users**: 회원 정보
- **products**: 제품 정보
- **orders**: 주문 정보
- **order_items**: 주문 상세 항목
- **cart**: 장바구니
- **reviews**: 리뷰
- **inquiries**: 1:1 문의
- **stores**: 매장 정보
